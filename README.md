# How to run

| #       | Description           | Command  |
| :------------- |:-------------| :-----|
| 1      | Install dependencies | `cd ${current.project.path} && bundle install` |
| 2      | Run | `cd ${current.project.path} && rails server -b 0.0.0.0` |
